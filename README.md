# Speech Emotion Recognition

A deep learning project that recognizes emotions from speech audio using a fine-tuned Wav2Vec2 model.

## Features

- Upload WAV audio files
- Predict 7 emotions:
  - Angry
  - Disgust
  - Fearful
  - Happy
  - Neutral
  - Sad
  - Surprised
- Real-time inference
- Streamlit web application

## Tech Stack

- Python
- PyTorch
- Hugging Face Transformers
- Wav2Vec2
- Librosa
- Streamlit

## Model

Fine-tuned Wav2Vec2 model trained on:
- RAVDESS
- TESS
- CREMA-D
